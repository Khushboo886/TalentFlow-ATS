# TalentFlow-ATS
TalentFlow ATS is an end-to-end Applicant Tracking System designed as a learning-first SaaS project.

The project focuses on building a real-world hiring platform while learning modern full-stack development practices including frontend development with React, backend APIs using FastAPI, database design with PostgreSQL, secure authentication using JWT and RBAC, resume parsing, hiring pipelines, and Google Sheets synchronization.

This repository follows a phase-wise learning and development approach, making it suitable for beginners who want to understand how production-grade SaaS systems are built.
